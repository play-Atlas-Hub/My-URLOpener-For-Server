# URLOpener
Welcome to this simple project in Python. This project allows one to open their set of urls in their browser. 


To run the project do the following steps:
1) Clone the repo, by running: ``` git clone https://github.com/extreme-top/URLOpener/ ```
2) ``` cd URLOpener ```
3) Using your favorite text editor, open website_automator.py and add/modify any urls. You can add more "plugin_packs" if you want.
3) Run the command ``` python3 website_automator.py ```
